# Screenshots needed

Take these screenshots with the app running (`uv run python -m miniagent`), then save them here.

| Filename | What to capture |
|----------|-----------------|
| `chat-streaming.png` | A response mid-stream — blinking cursor visible, partial text |
| `chat-done.png` | A completed response with rendered markdown (headers, bold, lists) |
| `browser-screenshot.png` | A response where Claude took a screenshot — the inline image visible in the bubble |
| `tool-indicator.png` | The ⚙ tool call indicator showing (navigate + screenshot steps) |
| `welcome.png` | The empty welcome screen before any message is sent |

**Tips:**
- Use macOS `⌘ Shift 4` to capture a region, or `⌘ Shift 5` for the window
- Crop tightly to the chat area, no need to include the browser chrome
- Aim for ~1200–1600px wide retina captures for GitHub rendering quality
- PNG format preferred
